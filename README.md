# 小鼠开颅手术前后智力评估研究项目

## 项目简介

本项目旨在研究小鼠在动开颅手术后智力水平的变化，通过实施动开颅手术并进行智力评估实验，我们将探讨手术对小鼠认知能力的潜在影响。

## 项目目录结构 📓

- `/data`：存放实验数据
- `/notebooks`：用于数据分析和实验记录
- `/scripts`：存放用于数据处理和分析的Python脚本
- `/results`：存放实验结果和可视化输出
